ModeToggle

Three-mode toggle button (Light / Dark / Vibrant). Cycles through modes on click.

Props

Prop Type Default
None - -

Usage

---
import ModeToggle from '../components/ui/ModeToggle.astro';
---

<ModeToggle />

Behavior

  • Reads window.modeSwitcher (initialized by mode-switcher.js)
  • Persists selection to localStorage.mode
  • Emits mode-change event on window
  • Updates data-mode attribute on <html>
